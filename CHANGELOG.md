# Changelog

All notable changes to ShellGate will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Core terminal access via WebSocket with xterm.js frontend
- PTY allocation with terminal resize support
- Token authentication (auto-generated by default)
- Password authentication with bcrypt and rate limiting
- TOTP two-factor authentication
- Self-signed TLS certificate generation (ED25519)
- Let's Encrypt auto-TLS via autocert
- Custom certificate support
- Session management with timeout and idle controls
- Session recording in asciicast v2 format
- Session sharing with read-only links and TTL
- IP whitelisting (CIDR ranges)
- Per-IP rate limiting
- Health check endpoint with uptime and session count
- CLI with cobra: serve, setup-otp, sessions, version, completion
- Embedded static assets (zero external files at runtime)
- Security headers (CSP, X-Frame-Options, etc.)
- Graceful shutdown on SIGINT/SIGTERM
- Cross-platform builds: linux, darwin, windows (amd64, arm64)
- GitHub Actions CI/CD with GoReleaser
- curl-pipe installer script
